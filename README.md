## おおまかな流れ
[Node.js](https://nodejs.org/en/)をインストール。  
インストールが完了したら以下のcommandを実行。
```bash
$ git clone このリポジトリ
$ cd this_project		#プロジェクトフォルダに移動
$ npm install gulp -g	#gulpをグローバルにインストールする
$ npm install			#npmプラグインをインストール
$ gulp					#これで動きます
```
